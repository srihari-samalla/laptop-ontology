Part of learning from the Winter School of Knoweldge Graph and Semantic Web 2026

I have developed a basic ontology for Laptop. The major entities included are the components, characteristics, types, laptop make, operating system, port, securing mechanism, sensor and make of laptop.

The files included in the repository are:

    The OWL ontology formalization in the RDF/XML format.
    Basic Lightweight ontology made using UML.
    Visualization of the SmartWatch ontology using OntoGraph.
